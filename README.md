# OutreachHomepage

Homepage for UoB Computer Science outreach activities for open days and educational events.

## About

This repository hosts a GitHub Pages site that serves as a central hub for the University of Bradford Computer Science outreach activities. The homepage provides:

- Links to interactive activities and educational resources
- Information about available programming, AI/ML, and cybersecurity challenges
- QR codes for quick access to resources during events

## Activities Featured

1. **Python Maze Game** - Interactive maze-solving using Python code
2. **AI & ML Jupyter Notebooks** - KS3/KS4 educational materials for AI and ML concepts
3. **Capture The Flag Challenge** - Cybersecurity challenge for students

## Viewing the Site

The homepage is available at: https://uob-cs-outreach.github.io/OutreachHomepage/

## Local Development

To view the site locally, simply open `index.html` in your web browser.
